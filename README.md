# Vacasky Landing Page

A modern, minimalist travel landing page built with HTML, Tailwind CSS, and optimized for a premium experience.

## Features
- **Modern Design**: Sleek, minimalist aesthetics with a focus on high-quality visuals.
- **Dark Mode Support**: Seamless toggle between light and dark themes.
- **Responsive Navigation**: Includes a mobile-friendly menu.
- **animations**: subtle fade-in and hover effects for a dynamic feel.
- **SEO Optimized**: Includes meta tags and semantic HTML structure.

## Getting Started

To run the project locally, you can use:

```bash
npm run dev
```

This will start a local development server.

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Outfit & Bebas Neue)
- Material Icons
